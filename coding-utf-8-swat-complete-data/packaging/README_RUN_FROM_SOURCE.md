# Run From Source (No EXE)

If you do not want to build an `.exe`, you can run the app directly:

```powershell
python swat_weather_app.py
```

Make sure required packages are installed in your Python environment:

- geopandas
- rasterio
- shapely
- numpy
- pandas
- xarray
- netcdf4
- imdlib
- openpyxl
- xlsxwriter
- requests
